## 资源
> 只需要：1核1G即可，暂时只有X86版本

## 安装与部署
```
docker run -itd --name udocker -p 8000:8000 -p 9002:9002 -v /var/run/docker.sock:/var/run/docker.sock  llody/udocker:v1.0-base
```
